1. Home JavaScript is used to program the behaviour of the web pages.
2. Introduction JavaScript can be used to alter HTML Content, Attributes and Styles. It can also be used to validate data input.
3. Where To JavaScript code must be inserted between script tags. A function is a bloack of JavaScript code executed when an event occurs. JavaScript code can be kept in the <head> tag, <body> tag or in an external file using <script src=""> </script>/code>.
4. Output JavaScript can be used to write in an alert box using windows.alert(), the HTML output using document.write();, an HTML element using innerHTML and the browser console using console.log();.
5. Syntax Syntax is the set of rules in which how programs are constructed. A statement has values, operators, expressions, keywords and comments. There are fixed values(literals) and variable values. Identifiers must start with _ or $.
6. Statements JavaScript Statements are separated witha semicolon. It ignores white space. A statement is broken if does not fit in one line. Code blocks are used to group statements.
7. Comments Single line comments start with //. Multi-line comments are written between /* and */.
8. Variables Variables are used to store data values. They must be identified with a unique name called Identifiers. A variable can be declared using this format var carName or var carName = "Volvo";
9. Operators JavaScript Operators include addition(+), subtraction(-), multiplication(*), division(/), modulus(%), increment(++) and decrement(--).

JavaScript Comparison Operators include equal to(==), equal value and equal type(===), not equal(!=), not equal value and equal type(!==), greater than or equal to(>=), less than or equal to(<=) and the ternary operator(?).
10. Arithmetic An arithmetic operation operates on two numbers called operands. The operation is defined by the operator.
